# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Как пользоваться
1. Скачиваете репозиторий к себе на компьютер
2. Открываете консоль и прописываете путь к директории с только что скачанным репозиторием следующей командой:
```
cd *Путь к директории*
```
3. У Вас должен быть установлен Python. Для корректного запуска сайта необходимо установить несколько библиотек. Сделать это можно в терминале с помощью команды:
```
pip install -r requirements.txt
```
4. После того как все необходимые библиотеки установлены, можно запустить сайт с помощью команды:
```
python main.py
```
5. Сайт запущен, попасть на него Вы сможете перейдя по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000)
