# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Как пользоваться
1. Скачиваете репозиторий к себе на компьютер
2. Открываете консоль и прописываете путь к директории с только что скачанным репозиторием следующей командой:
```
cd *Путь к директории*
```
3. У Вас должен быть установлен Python. Для корректного запуска сайта необходимо установить несколько библиотек. Сделать это можно в терминале с помощью команды:
```
pip install -r requirements.txt
```
4. После того как все необходимые библиотеки установлены, у Вас должен быть составлен excel файл для подгрузки данных о винах на сайт в следующем формате:
<img width="1410" height="260" alt="image" src="https://github.com/user-attachments/assets/789d75d8-f59f-4064-8029-e8373c3ec8cb" />

5. Теперь Вы можете запустить сайт. Скрипт требует указания имени excel файла(если он лежит в одной директории со скриптом) или пути к этому файлу, указывать при этом с расширением файла. Сделать это можно двумя способами - через консоль и через виртуальное окружение. Вариант через консоль: 
```
python main.py -p=*имя или путь файла*
```
Или просто создайте в директории со скриптом файл `.env` и впишите туда путь к нужному excel файлу в переменную `EXCEL_PATH=*путь*`. В таком случае Вы сможете запустить сайт командой:
```
python main.py
```
Также у скрипта есть несколько опциональных параметров, которые не обязательно указывать:
- s (sheet_name) - имя листа в файле excel. Стандартно скрипт подгружает первый лист из полученного файла. Если в Вашем файле много листов, скрипту необходимо указать имя листа, из которого брать данные. Этот параметр можно передать скрипту записав его в `.env` занеся в переменную `SHEET_NAME=`.
- y (creation_year) - год создания винодельни. Можете указать данный параметр, если хотите поменять возраст винодельни отображающийся на сайте. Также этот параметр можно передать скрипту записав его в `.env` занеся в переменную `CREATION_YEAR=`.

Если Вы не указали нужные параметры в переменном окружении `.env`, тогда необходимо передать их вручную через консоль следующим образом:
```
python main.py -s=List1 -y=2010
```
Более подробную информацию по параметрам можно получить с помощью команды:
```
python main.py -h
```
> **Примечание**: если сайт запустится Вы не получите в консоли никакого оповещения. Сайт будет запущен, пока Вы не выключите компьютер или пока самостоятельно не выключите сервер. Принудительно выключить сайт можно в терминале с помощью комбинации клавиш `Ctrl+C`.
6. Сайт запущен, попасть на него Вы сможете перейдя по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000)
